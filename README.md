Aqui está uma breve descrição dos exercícios de KNN e DMC propostos na aula. 
-> O exercício 1 basicamente é a implementação do NN quando k = 1
-> O exercício 2 é a mesma coisa que o KNN, só que com k sendo 1, 3, 5, 7 e 9 
-> O exercício 3 é a implementação manual do DMC e da Matriz Confusão
-> Todos eles exibem gráficos em relação à Taxa de Acerto Médio, incluindo no exercício 3 a exibição da Matriz Confusão

== PROCESSO PARA EXECUTAR OS CÓDIGOS ==
1. Clonar o repositório para outra máquina
2. python -m venv venv
3. .\venv\Scripts\activate
4. pip install -r requirements.txt (para instalar tudo o que precisar)
5. Para rodar no terminal os códigos caso precise:
    python exercicio1.py
    python exercicio2.py
    python exercicio3.py
